AI Sales Intelligence Platform
An agentic AI platform that researches any B2B prospect, scores closing probability, calculates sentiment and urgency, and generates a personalized three message outreach sequence automatically.
What it does

Searches the web for real company intelligence using Tavily
Analyzes product screenshots using computer vision
Calculates closing probability, sentiment score, urgency score, and predicted response rate
Aligns prospect pain points with your company strengths using a RAG knowledge base
Generates three personalized outreach messages using the soft sell strategy
Logs everything to HubSpot CRM automatically

Production numbers from Day 1 testing

Pipeline completes in 16 to 41 seconds
Finds 9 web sources per prospect
Closing probability scored at 70 percent on Notion test
Three personalized messages generated per prospect
Cost per prospect: $0.00 using free tiers

Tech stack

LangGraph for agent orchestration
Groq with Llama 3 for free LLM inference
Tavily for real time web research
ChromaDB for knowledge base RAG
HubSpot free CRM for contact management
LangSmith for eval and monitoring
FastAPI for backend API
React for frontend
